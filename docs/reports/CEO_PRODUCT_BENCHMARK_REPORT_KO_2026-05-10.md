## 6. 유료 발행 차단 기준

| 차단 조건 | 설명 |
| --- | --- |
| 출처가 약함 | factual claim을 뒷받침하지 못하면 유료 발행 금지 |
| 법률 리스크 높음 | 투자자문, 과장광고, 저작권 리스크 있으면 금지 |
| 고객별 가치 없음 | 모두에게 같은 요약이면 유료 상품으로 부족 |
| Watchlist 없음 | 고객이 다음에 무엇을 봐야 하는지 모르면 부족 |
| 숫자/표/시나리오 없음 | 판단 보조가 아니라 읽을거리 수준에 머무름 |
| 무료 인터넷 글과 차이 없음 | 결제 이유가 없음 |
