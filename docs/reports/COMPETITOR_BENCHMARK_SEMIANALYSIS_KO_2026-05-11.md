## 1. Executive Summary

Harness가 물리 AI 정보 구독 사업에서 참조해야 할 No.1 경쟁사는 **SemiAnalysis**다.

이 회사는 반도체·AI 인프라 분야에서 "뉴스레터"로 시작해 **연매출 $100M+ 연구 기업**으로 전환한 유일한 사례다. 창업자 Dylan Patel은 반도체 학위 없이 2020년 혼자 시작해, 6년 만에 85명·11개국 팀, 250,000+ 구독자, $100M 매출 궤도에 올려놨다.

**Harness에 대한 핵심 메시지는 세 가지다.**

1. 뉴스레터는 수익 채널이 아니라 **영업 깔때기**다. 진짜 돈은 institutional research에서 나온다.
2. 콘텐츠 깊이가 전부다. SemiAnalysis 기사 하나는 평균 **8,000~9,000단어, 차트 10개 이상**이다. Harness 현재 수준(3~5문장)과의 격차는 구조적이다.
3. Korean × Physical AI 포지션은 여전히 유효하다. SemiAnalysis가 커버하지 않는 **한국어·한국 제조·물리 AI 현장 맥락**은 Harness만의 실제 whitespace다.

## 3. 수익 구조 전체 해부

### 3.1 수익 스택

SemiAnalysis의 수익 모델은 단순한 유료 뉴스레터가 **아니다**.
실제 구조는 3개 레이어로 작동한다.

```
Layer 1 — 뉴스레터 (Free + $500/year paid)
  └→ 역할: 독자 확보 + 브랜드 + 리드 제너레이션
  └→ 수익 기여: 전체의 소수 비율 (창업자 표현: "charitable project")

Layer 2 — Institutional Research & Models
  └→ 역할: 핵심 수익원
  └→ 제품: 8+ 독점 산업 모델, ChipBook 월간 트래커, Core Research 스트림
  └→ 고객: 기술 대기업, VC·헤지펀드, 정부기관
  └→ 가격: 비공개 (연간 수백만 달러 계약 추정)

Layer 3 — Consulting & Advisory
  └→ 역할: 고마진 부가 수익
  └→ 형태: retained advisory, bespoke project, monthly analyst calls
  └→ 추가: 창업자의 ~20개 스타트업 지분 (정보 우위 → 투자 수익)
```

### 3.2 각 레이어 상세

#### Layer 1: 뉴스레터

- 공개 무료 기사: 주요 산업 분석 글, 완전 공개
- 유료 구독 $500/year: 더 깊은 분석 접근, 아카이브
- Substack 250,000+ 구독자 중 유료 전환율은 비공개이나, 1~3% 전환 가정 시 2,500~7,500명 × $500 = **$1.25M~$3.75M**
- **이것이 전체 매출의 5% 미만으로 추정**

#### Layer 2: Institutional Research Models

보유 모델 목록 (전부 기관 전용, 가격 비공개):

| 모델명 | 커버 영역 |
|---|---|
| Accelerator & HBM Model | Nvidia/AMD/Google TPU SKU별 출하량, ASP, 수익 |
| AI Cloud TCO Model | GPU $/시간, 학습 $/PFLOP, 추론 $/M tokens |
| AI Networking Model | 스위치·광학·케이블 인프라, 80개 하이퍼스케일러 분석 |
| Datacenter Industry Model | 5,000개 시설 IT 전력·용량, 위성이미지 기반 |
| Foundry Industry Model | TSMC 고객·노드·웨이퍼 출하, 2030년 분기별 예측 |
| Tokenomics Model | 하드웨어→소프트웨어 ROI, 모델 아키텍처 벤치마킹 |
| Wafer Fab Equipment Model | EUV/ArFi/KrF 리소그래피 스택, 2027년 판매 예측 |

→ **이 모델들이 $100M 매출의 핵심이다.**

#### Layer 3: Consulting

- Retained advisory: 특정 기업/투자자를 위한 상시 자문
- Bespoke project: 맞춤형 딥다이브 리서치

- 지분: ~20개 스타트업 equity (정보 우위의 재투자)

## 4. 콘텐츠 전략 해부

### 4.1 기사 한 편의 구조

SemiAnalysis 기사는 **"단순 요약"이 아니라 의사결정 자산**이다.

실측 분석 (AI Value Capture - The Shift To Model Labs 기준):

| 항목 | SemiAnalysis | Harness 현재 |
|---|---|---|
| 평균 글자 수 | 8,000~9,000단어 | 3~5문장 (~300자) |
| 차트/테이블 수 | 10개 이상 | 0개 |
| 독점 모델 링크 | 6개 이상 | 없음 |
| 정량적 근거 | Anthropic ARR $9B→$44B, 토큰 실가격 $0.99 등 | 없음 |
| 유료 게이트 위치 | 3개 섹션에서 설정 | 없음 |
| 필자 수 | 4명 팀 공동 작성 | AI 단독 생성 |

### 4.2 SemiAnalysis가 독자에게 주는 것

창업자 Dylan Patel 표현: **"모두가 알기 전에 먼저 아는 것"**

구체적으로는:

1. **가격책정 프레임워크**: GPU 렌탈, 추론 비용, 칩 BOM의 이론 상·하한을 명확히 제시 → C-suite가 협상에 즉시 사용
2. **공급망 인텔리전스**: TSMC 가동률, HBM 수급, 파운드리 용량을 경쟁자보다 먼저 예측
3. **투자 기회 지도**: "메모리 벤더가 수혜자", "추론 업체 마진 >70%" 등 포지션 명시
4. **독점 모델 접근**: 내부 분석팀이 수개월 작업할 데이터를 구독으로 획득

### 4.3 유료 전환 심리

SemiAnalysis가 독자를 유료로 전환하는 3가지 메커니즘:

1. **불완전한 공개**: 무료 기사에서 핵심 데이터를 Paywall 뒤에 배치
2. **반복 모델 참조**: 기사마다 institutional model 6개 이상 링크 → "저것도 구독해야겠다" 유도
3. **B2B 표적 설계**: Nvidia·Microsoft 의사결정자들이 실명 언급 → 동료 압박 효과

### 6.2 그러나 직접 비교는 잘못된 프레임이다

SemiAnalysis가 Harness의 목표가 아니다. **2014년의 Stratechery가 Harness의 현실적 목표**다.

Ben Thompson은 혼자 시작해 40,000+ 구독자, 연 $5M+을 달성했다. 그는 3가지를 지킨 것뿐이다:
- 주 3회 이상 발행 일관성
- 독자적 분석 프레임워크(Aggregation Theory)
- 구독자와의 신뢰 관계 복리 구축

Harness가 Korean Physical AI 공간에서 이것을 달성할 수 있다면 충분히 성공이다.

### 6.3 Harness가 실제로 채울 수 있는 Whitespace

| 격차 항목 | SemiAnalysis의 한계 | Harness의 기회 |
|---|---|---|
| 언어 | 영어 전용 | 한국어 독자 전용 포지션 |
| 지역 맥락 | 미국 중심 | 한국 제조·반도체·현장 함의 |
| 독자층 | 업계 전문가 (영어권) | 한국어 관심 독자 (비전문가 포함) |
| Physical AI 현장 | 공급망 재무 중심 | 현장 적용, 직업 변화, 한국 기업 임팩트 |
| 접근성 | 기술적 밀도 높음 | 비전문가도 읽을 수 있는 해석 |

## 7. Harness가 즉시 적용해야 할 인사이트

### 인사이트 1: 뉴스레터는 영업 깔때기로 재정의하라

SemiAnalysis의 뉴스레터는 "창업자 표현으로 charitable project"다. 수익이 목적이 아니라 고객을 끌어오는 수단이다.

Harness도 무료 이슈를 "구독자 확보 + 신뢰 구축 수단"으로 명확히 재정의해야 한다. 유료 전환은 가치 증명 이후다.

### 인사이트 2: 콘텐츠 깊이를 최소 10배 높여라

현재 Harness Tier 3 Refiner는 3~5문장을 생성한다. 이것은 신호 요약이지 지식 자산이 아니다.

최소 기준:
- 기사당 1,500~2,000단어
- 1개 이상의 수치 테이블 (비용, 시장 규모, 기업 비교)
- 1개 watchlist block (다음 주 추적 대상)
- 1개 decision block ("한국 독자에게 무엇을 의미하는가")

### 인사이트 3: 가격을 올려라

₩9,900/월 (~$7.3)은 Stratechery $150/년 ($12.5/월)보다 낮다. 콘텐츠 품질이 받쳐준다면 ₩14,900~₩19,900도 한국 시장에서 방어 가능하다.

단, 지금 올리는 것이 아니다. 먼저 품질이 올라가야 한다.

### 인사이트 4: 데이터 자산을 최소 1개 구축하라

SemiAnalysis의 핵심 moat는 독점 모델(Accelerator Model 등)이다.

Harness의 Phase 2 목표는 "한국 Physical AI 기업 트래킹 매트릭스" 1개 구축이다. 예:
- 한국 로봇·자동화 관련 기업 50개 반기별 동향 추적
- 국내 제조 자동화 도입 사례 DB

### 인사이트 5: 분석 프레임워크를 1개 만들어라

Stratechery = Aggregation Theory
SemiAnalysis = Supply Chain Forensics + TCO Framework

Harness가 필요한 것: **"Physical AI 한국 실익 프레임워크"**

예: "이 기술 신호가 한국의 제조/반도체/서비스/일자리 중 무엇을 어떻게 바꾸는가"를 매주 동일 기준으로 적용하면 독자는 예측 가능한 분석 렌즈를 갖게 된다.
