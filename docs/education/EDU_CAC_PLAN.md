# 교육 컨설팅 고객 획득 비용(CAC) 계획

> 작성: 2026-05-24 | AR-029 Red Team 조건 B 해소 | correlation_id: edu-consulting-20260524
> 목적: 첫 50명 익명 학부모 발굴 채널·비용·타임라인 구체화

---

## 1. 채널 선택 원칙

- 부대표 또는 대표의 지인망에 직접 결제 요청 금지 (CLAUDE.md §Never)
- 수익은 마케팅을 통해 유입되는 **익명 고객**에서만 인정
- CLAUDE.md §5 Must: 외부 발행·광고 전 `legal_review_approve` + `qa_clear` 필수
- 표시광고법: "베타 참여" "사전 관심 조사" 형식으로 운영 (과장 광고 금지)

---

## 2. 채널별 CAC 시뮬레이션

### 채널 A — 인스타그램 / 페이스북 유료 광고

| 항목 | 값 |
|---|---|
| 타겟 | 30~45세 초·중등 자녀 학부모, 교육 관심 |
| 예산 | $20~30 (pretotyping 1회) |
| 예상 도달 | 1,000~3,000명 |
| 클릭률 목표 | 2~5% → 20~150 클릭 |
| 랜딩→결제 버튼 클릭률 | 10~20% → **2~30 WTP 측정** |
| 예상 CAC | $1~15/명 (WTP 측정 기준) |
| 법률 제약 | 표시광고법 — "베타 참여" 문구 + QA `qa_clear` 필수 |
| 타임라인 | Pretotyping용 랜딩 완성 후 즉시 가능 (Red Team clear 선행) |

**적합성:** Pretotyping(WTP 측정)에 가장 빠른 채널. 결제 페이지가 아니라 "사전 관심 등록" 랜딩으로 운영하면 표시광고법 리스크 최소화.

---

### 채널 B — 맘카페·육아 커뮤니티 (Organic)

| 항목 | 값 |
|---|---|
| 타겟 | 네이버 맘카페, 베이비트리, 맘스홀릭, 클리앙 육아 게시판 |
| 비용 | $0 (글 직접 게시) |
| 예상 도달 | 100~500명 (게시글 당) |
| 법률 제약 | **네이버 ToS §9 자동화 크롤 금지** — 유기적 게시는 ToS 위반 아님 |
| 제약 사항 | 광고성 글 금지 커뮤니티 다수 — "설문 참여 요청" 형식으로만 가능 |
| CAC | $0 (시간 비용만) |
| 타임라인 | 콘텐츠 QA 완료 후 가능 |

**적합성:** WTP 인터뷰(5~10명 모집)에 적합. 직접 결제 유도 금지, "연구 참여자 모집" 형식으로 접근.

---

### 채널 C — 유튜브 쇼츠 / 릴스 (Organic SEO)

| 항목 | 값 |
|---|---|
| 타겟 | "AI 교육 불안" "스마트폰 중독" 키워드 검색 학부모 |
| 비용 | $0 (제작 시간 비용만) |
| 예상 도달 | 초기 100~1,000 뷰 (SEO 누적 필요) |
| 전환율 | 0.5~2% (쇼츠→프로필 링크) |
| CAC | $0 (첫 6개월 효과 낮음) |
| 타임라인 | 6~12주 이후 효과 발현 |

**적합성:** 장기 브랜드 구축에 유리. 단기 50명 모집엔 부적합. Phase 1 이후 병행.

---

### 채널 D — VP 소셜 / 지인 관찰 활용 (Product Insight만)

| 항목 | 값 |
|---|---|
| 활용 방식 | 부대표 생활권 관찰 → Pain Point 수집 (결제 요청 금지) |
| 수익 인정 | 금지 (CLAUDE.md §Never: 지인망 직접 결제 요청 불가) |
| 허용 활용 | 인터뷰 5~10명 모집 (무료 beta 참여, 결제 없음) |

**적합성:** WTP 인터뷰 대상 모집에 한해 활용 가능. 결제 유도 금지.

---

## 3. 50명 달성 타임라인

| 단계 | 기간 | 목표 | 채널 | 비용 |
|---|---|---|---|---|
| Pretotyping | 2주 | WTP 클릭률 측정 | 인스타 광고 | $20~30 |
| Pilot 모집 | 4주 | 유료 5~10명 | 인스타 광고 + 맘카페 | $30~50 |
| Growth | 8주 | 무료 50명 (이메일 리스트) | 콘텐츠 SEO + 입소문 | $0~20 |
| **누적** | **14주** | **50명 도달 목표** | | **$50~100** |

---

## 4. CAC 목표값

| 지표 | 목표 |
|---|---|
| Pretotyping CAC (WTP 측정) | < $15/명 |
| Pilot 유료 CAC | < $10/명 |
| 3개월 평균 CAC | < $5/명 (organic 증가 가정) |
| LTV (12개월 × ₩29,900) | $261 |
| LTV/CAC 목표 비율 | > 10x |
