# 사주·타로 서비스 벤치마킹 — Harness 온보딩 설계
## "역술인이 고객을 사로잡는 방식을 AI 교육 컨설팅에 이식한다"

> 작성: 2026-05-25 | correlation_id: edu-consulting-20260524
> CEO 지시: "사주나 타로점을 보러 갔을 때 역술인이 고객을 상대하는 방식과 동일하게 진행할 것"
> 조사 대상: 점신(1,800만 이용) · 포스텔러 · 마이파이 · Co-Star · Kasamba · Keen · 역술 심리학

## 1. 왜 사주/타로인가 — Harness와의 심리적 동형성

| 사주/타로 방문 고객 심리 | Harness 잠재 고객 심리 |
|---|---|
| "뭔가 잘못되고 있는 것 같아" | "AI 시대에 뭔가 놓치고 있는 것 같아" |
| "어떻게 해야 할지 모르겠어" | "AI 써야 하는지, 어떻게 써야 하는지 모르겠어" |
| "혼자 판단하기 두려워" | "내 아이에게 맞는 기준을 모르겠어" |
| "내 상황만 딱 맞는 답이 필요해" | "일반 강의 말고, 우리 애 케이스만 알고 싶어" |
| "검색해봤지만 내 얘기가 없어" | "유튜브 다 봤는데 우리 상황엔 안 맞아" |
| "믿을 수 있는 사람이 있으면 좋겠어" | "데이터로 설명해주는 전문가가 필요해" |

**결론:** 두 서비스의 고객은 동일한 심리 구조를 가지고 있다.
- **불확실성 + 개인 맞춤화 욕구 + 전문가 신뢰 욕구**

## 2. 역술인의 상담 7단계 — 심리 메커니즘 분석

### 2-1. 포착 (Observation) — 말하기 전에 이미 정보 수집

```
실제 역술인:
고객이 들어오는 순간 → 옷차림·표정·걸음걸이·손 상태·눈빛 관찰
→ "이 분은 최근 스트레스가 많은 분이다" 내부 판단 완료

Harness 온보딩 적용:
사용자가 첫 화면에서 클릭하는 패턴 관찰
→ "AI가 걱정돼요" vs "AI를 배우고 싶어요" 중 무엇을 먼저 클릭하는가
→ 클릭 순서, 망설임, 이탈 후 재진입 등 행동 데이터가 이미 프로파일
```

### 2-2. 진입 허들 최소화 (Ease of Entry)

```
실제 역술인:
"언제 태어나셨어요?" — 단 하나의 질문으로 시작
부담 없는 첫 질문 → 고객의 경계심 해제

한국 앱 패턴:
점신: 가벼운 무료 운세 → 자연스럽게 상담사 연결
포스텔러: 무료 콘텐츠 먼저 → 원하면 유료 확장
→ 공통: "일단 한 번 해봐" 진입 장벽 제거

Harness 적용:
첫 화면 질문 단 3개 (90초 이내 완성)
"결제 없이 진단 먼저 받아보세요"
```

### 2-3. 고확률 진술 (Barnum/Forer Effect)

이것이 가장 핵심적인 기법이다.

```
심리학적 정의:
Barnum/Forer Effect — 모호하지만 개인적으로 느껴지는 진술에
사람들은 "이건 나에 대한 얘기"라고 강하게 반응한다.

역술인 실제 발화:
"최근에 중요한 결정을 앞두고 계신 것 같아요."
→ 거의 모든 사람에게 해당하지만, 듣는 사람은 "어떻게 알았지?" 반응

Harness 학부모 버전 고확률 진술 예시:
"중학교 자녀를 두신 분들이 대부분 이런 고민을 하세요.
 'AI를 쓰면 사고력이 떨어질까봐', 동시에 '안 쓰면 뒤처질까봐'.
 두 걱정이 동시에 드신 적 있으시죠?"
→ 90%의 학부모에게 정확히 해당. 하지만 듣는 사람은 "나만 이랬나?" 느낌.

Harness 직장인 버전 고확률 진술 예시:
"팀에서 누군가가 AI로 뭔가를 빠르게 해냈을 때,
 '나만 뒤처지는 건 아닌가' 하는 느낌 받으신 적 있으세요?"
→ 거의 모든 직장인에게 해당.
```

**Barnum Effect 작동 조건 (Harness가 반드시 지켜야 할 것):**
1. 긍정·부정이 섞여야 함 ("잘 하고 계시지만, 이 부분에서...")
2. 구체적인 숫자나 상황이 포함돼야 함 ("중학교 1학년" 같은 맥락)
3. "대부분이" → "특히 000 같은 상황에서는" 순서로 이동

### 2-4. 반응 기반 조정 (Client Feedback Loop)

```
역술인 기법:
진술 → 고객 반응 관찰 → 맞으면 강화 / 틀리면 빠르게 전환

실제 대화:
역술인: "요즘 집안에서 갈등이 좀 있는 것 같은데요?"
고객: "음... 사실 직장 쪽이 더 문제예요."
역술인: "아, 사주상으로 직장 궁이 특히 요동치고 있어요.
         그게 가족에도 영향을 주는 거예요."
→ 틀린 것을 빠르게 회수하고 맞는 방향으로 통합

Harness 디지털 버전:
선택지 기반 분기 질문
"다음 중 더 와닿는 상황은?"
  A. "아이가 ChatGPT로 숙제를 그냥 베껴요"
  B. "아이가 AI 없이는 아무것도 못 하는 것 같아요"
  C. "학교에서는 쓰지 말라는데, 집에서는 어떻게 해야 할지"
  D. "다른 아이들은 AI로 뭔가를 한다는데 우리 애만 뒤처지는 것 같아요"
→ 고객이 선택 = 스스로 pain point 특정
→ 이후 처방이 자동으로 개인화
```

### 2-5. 고객이 정보를 채우게 만들기 (Client Participation)

```
역술인 핵심 기법:
"최근에 누군가와 관계가 어떠셨나요?" (모호한 질문)
→ 고객이 구체적 상황을 말함
→ 역술인은 그것을 가져다가 "사주에도 그게 나와 있어요"라고 돌려줌

심리학: 사람들은 자신이 말한 것을 남이 알아챈 것처럼 느낀다

Harness 적용:
"어떤 상황이 가장 걱정되세요? 자유롭게 말씀해주세요."
→ 고객이 말한 것을 Claude가 분류 후 처방에 그대로 반영
→ 처방에 "말씀하신 것처럼 [자녀가 ChatGPT로 답 베끼는 상황]에서..."
→ 고객: "어떻게 내 상황을 이렇게 정확히 알지?"
```

### 2-6. 처방의 부분 공개 (Teaser → Full Reveal)

```
역술인 기법:
"운세상 이 방향이 맞긴 한데..."
"이 부분은 조금 더 자세히 봐야 해요"
→ 다음 방문, 추가 상담 유도

실제 앱 패턴:
Kasamba: 3분 무료 → 이후 유료
Keen: 5분 $1 → 이후 정가
포스텔러: 무료 콘텐츠 → 유료 상세 리포트
마이파이: 기질 분석 무료 → 상세 직업 적합도 유료

Harness 적용:
무료 진단 결과: "[아이 유형] + 처방 2가지 (무료)"
유료 전환 포인트: "이 경우 더 중요한 3번째 요소는..."
→ 첫 결제: 원포인트 레슨 ₩9,900 (정가 ₩49,000)
→ 또는: "4주 부모 AX 과정 무료 체험 2주"
```

### 2-7. 레이블링과 소속감 (Labeling & Identity)

```
역술인 기법:
"000님은 ○○띠라서 이런 특성이 있어요"
"이 사주에는 특별한 기운이 있어요"
→ 사람들은 레이블을 받으면 그 레이블에 맞게 행동하고,
   그 레이블을 내 것으로 받아들임

성공 사례:
Co-Star: 별자리 기반 정밀 성격 분석 → "나 INFJ야"처럼 공유
MBTI: 레이블 자체가 소셜 콘텐츠가 됨

Harness 레이블링 시스템:
[부모 유형]
- "AI 이중 공포형" — AI 쓰면 걱정, 안 쓰면 더 걱정
- "AI 관망형 부모" — 지켜보는 중이지만 기준이 없음
- "AI 주도형 부모" — 먼저 배우려 하지만 어디서 시작해야 할지 모름

[아이 유형]
- "정답 바이패스 유형" — ChatGPT로 답만 복사
- "AI 의존 발아 단계" — 아직 초기지만 패턴 고착 전 골든 타임
- "AI 협업 성숙 단계" — 올바른 사용 중, 고도화 필요

→ 이 레이블이 바이럴 콘텐츠가 됨 ("우리 아이 AI 의존 유형 알아보기")
→ SNS 공유 → 유입 루프
```

## 4. Harness 온보딩 설계 — "사주 모델" 완전판

### Step 0: 포착 (Observation)
고객이 랜딩 페이지에서 보이는 행동 데이터 수집
- 어떤 카피에서 멈추는가 (학부모 vs 직장인)
- 어떤 버튼을 먼저 클릭하는가 (AI 불안 vs AI 배우기)
- 이탈 후 재방문 패턴

### Step 1: 비침습적 3문항 (90초)

```
Q1. [세그먼트 분류 — 역술인의 "생년월일"]
    저는...
    ○ 초·중·고 자녀를 둔 부모
    ○ AI 압박을 받는 직장인
    ○ AI 시대 커리어가 걱정되는 직장인
    ○ 부모님의 AI 소외가 걱정되는 자녀

Q2. [Pain Point 분류 — 역술인의 "오늘 무슨 일로 오셨어요"]
    지금 가장 걱정되는 것은?
    ○ 아이가 AI에 너무 의존하는 것 같아요
    ○ 아이가 AI를 너무 안 써서 뒤처질까봐
    ○ 무엇이 옳은지 기준이 전혀 없어요
    ○ 나 자신이 AI를 모르니까 아이를 가이드 못 해요 ← (부모 먼저 원칙)

Q3. [온도 파악 — 역술인의 "얼마나 급하신가요"]
    AI에 대해 직접 써보신 경험이 있으신가요?
    ○ 거의 없어요 (이제 막 시작)
    ○ 가끔 써봤어요 (아직 익숙하지 않음)
    ○ 꽤 써봤어요 (쓰는데 뭔가 부족한 느낌)
```

### Step 2: "맞아요 순간" 연출 (Barnum Effect)

**입력값 → 고확률 진술 자동 생성 (LLM)**

```
예시 — Q1: 학부모 / Q2: 기준이 없음 / Q3: 가끔 써봤음

출력:
"[아, 저도 잘 알 것 같은 상황이에요]

자녀가 AI를 쓰는 걸 봤을 때,
혼내야 할지 내버려둬야 할지 판단이 안 서신 적 있으시죠?

주변 엄마들은 "AI 시켜야 한다", "AI가 망친다"로 나뉘는데
정작 우리 아이에게 뭐가 맞는지는 아무도 안 알려주는 것 같고.

그 답답함, 완전히 이해합니다.
사실 그게 당연한 이유가 있어요."

→ 고객이 느끼는 것: "어떻게 내 상황을 이렇게 정확히 알지?"
→ 실제: 이 상황은 90%의 학부모에게 해당하는 고확률 진술
```

### Step 3: 고객이 정보를 채우게 만들기

```
"조금 더 여쭤봐도 될까요?"

"최근에 아이가 AI를 사용한 구체적인 상황이 있으셨나요?
 (예: 숙제에 사용, 질문에 사용, 창작에 사용 등)"
                   ↓ 자유 기술 or 선택지
"어떤 반응이 가장 걱정되셨나요?"
 ○ 아이가 AI 없이는 시작을 못 해요
 ○ AI 답변을 그대로 제출해요
 ○ 아이가 AI를 어떻게 쓰는지 제가 모르겠어요
 ○ 선생님이 AI 쓰지 말라고 했는데 몰래 써요
```

### Step 4: 무료 처방 일부 공개 (Teaser)

```
"000님 상황 분석 완료"

━━━━━━━━━━━━━━━━━━━━━━
[진단 결과]
아이 유형: AI 의존 발아 단계 ⚠️
부모 유형: AI 기준 부재형 📋
━━━━━━━━━━━━━━━━━━━━━━

지금 상황에서 가장 중요한 것 3가지:

✅ 1. [오늘 당장 할 수 있는 것]
   "AI를 왜 쓰는지 아이에게 한 번 물어보세요."
   이 질문만으로도 아이의 인식이 달라집니다.

✅ 2. [이번 주 안에 해야 할 것]
   "이 유형의 아이에게는 이렇게 접근하면..."
   ← 이 부분은 상황별로 다르게 설계해야 해서
      맞춤 처방에서 자세히 설명드릴게요

🔒 3. [가장 중요한 핵심 — 이게 안 되면 1, 2도 무효]
   "000님 아이에게 특히 맞는 접근법은..."
   ← 이 부분은 000님 아이의 구체적 상황을 더 알아야 해요
━━━━━━━━━━━━━━━━━━━━━━

[지금 바로 받아보실 수 있어요]
→ 맞춤 상담 첫 번째: ₩9,900 (정가 ₩49,000)
   "오늘 받아보신 진단을 바탕으로 000님 아이만을 위한 처방"
또는
→ 4주 부모 AX 과정 2주 무료 체험
   "부모님이 먼저 AI를 이해한 후 아이를 가이드하는 법"
```

### Step 5: 레이블 + 공유 유도 (바이럴 설계)

```
카드 형태로 공유 가능:

┌─────────────────────────┐
│  📋 나의 AI 부모 유형   │
│  ──────────────────── │
│  AI 기준 부재형         │
│  "알고 싶지만           │
│   어디서 시작할지       │
│   모르는 부모"         │
│                          │
│  우리 아이 유형:         │
│  AI 의존 발아 단계 ⚠️   │
│  → 지금이 골든 타임     │
│  ──────────────────── │
│  by Harness            │
└─────────────────────────┘

[내 결과 공유하기] → 카카오톡·인스타 → 바이럴
[자세한 처방 받기] → 유료 전환
```

## 5. 핵심 카피 원칙 (역술인 언어를 Harness에 이식)

| 역술인 언어 | Harness 번역 |
|---|---|
| "사주상으로 이 시기가..." | "연구에 따르면 이 나이 단계에서는..." |
| "당신의 기운이..." | "000님 아이의 학습 패턴이..." |
| "이건 당신만의 문제가 아니에요" | "이런 고민을 하는 부모님이 전국에 수백만 명이에요" |
| "지금 이 시기가 중요해요" | "이 시기는 AI 의존 패턴이 굳어지기 전 골든 타임이에요" |
| "다음에 한 번 더 오세요" | "3개월 후 어떻게 달라졌는지 점검해드릴게요" |
| "이 방향이 맞긴 한데..." | "처방 방향은 맞아요. 다만 000님 아이의 경우 특히..." |

## 6. Harness가 훼손하면 안 되는 본질

사주/타로를 벤치마킹하되, 아래는 절대 훼손하지 않는다:

| 사주/타로 방식 (하지 않을 것) | Harness 본질 (지킬 것) |
|---|---|
| 근거 없는 운명론 ("당신의 운이...") | 연구 기반 증거 인용 |
| 공포 조장 ("이걸 안 하면 큰일 나요") | 손실 언어는 쓰되, 과장 없이 |
| 결과의 불확실성 덮기 | "이 방법이 100% 효과 있다"는 보장 금지 |
| 무한 방문 유도 (의존 조장) | 자립 목표: "이것을 알면 혼자 판단 가능" |
| 모호한 일반 처방 | 구체적·실행 가능한 처방 |
| 소수에게만 유효한 클로즈드 정보 | 연구는 공개된 것, Harness는 해석을 파는 것 |

**핵심 차이:**
사주/타로는 "나만 아는 신비한 정보"를 파는 척한다.
Harness는 "공개된 연구를 당신 상황에 맞게 번역"하는 것을 판다.

방식은 사주를 따르되, 내용은 과학을 따른다.

## 7. A/B 테스트 우선순위

1. **Step 2 Barnum 진술 버전 비교:**
   - A: "많은 부모님들이..."
   - B: "중학교 1학년 자녀를 둔 부모님들이..."
   → 구체적 맥락이 Barnum Effect를 강화하는지 검증

2. **Step 4 Teaser 강도 비교:**
   - A: 처방 1개 무료 + 2개 잠금
   - B: 처방 0개 + 진단만 무료
   → 처방 일부 공개가 전환율에 미치는 영향

3. **Step 5 레이블 공유율 비교:**
   - A: 부모 유형 레이블
   - B: 아이 유형 레이블
   → 어떤 레이블이 더 많이 공유되는가

4. **첫 결제 구조 비교:**
   - A: 원포인트 레슨 ₩9,900 (Keen 모델)
   - B: 2주 무료 후 구독 ₩29,900 (Free Trial 모델)
   → LTV 기준으로 비교
