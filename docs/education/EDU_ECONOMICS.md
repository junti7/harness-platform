## 2. 가격 구조 설계 — "잃어버려도 괜찮은 금액" 원칙

### 핵심 심리 기준 — 한국 마이크로 결제 행동 벤치마크

> **CEO 2026-05-25:** "카카오톡 유료 이모티콘 몇천원, 네이버 웹툰 미리보기 몇천원, 게임 아이템 몇천원 구매하는 심리와 비슷한 것."

한국 소비자는 이미 마이크로 결제 생태계에 익숙하다. 이 패턴을 Harness가 그대로 탑승한다.

| 레퍼런스 | 가격대 | 구매 심리 | Harness 적용 |
|---|---|---|---|
| 카카오 이모티콘 | ₩2,200~₩3,300 | "이거 귀여워서 그냥 삼" — 망설임 0 | Step 1~3 묶음 = ₩2,970 |
| 네이버 웹툰 쿠키 (미리보기 1화) | ₩200~₩500/화 | "이 다음 내용이 너무 궁금해" | Step 1 무료 → "다음 패턴 분석이 궁금해" |
| 카카오게임 아이템 | ₩1,100~₩5,500 | "이거 사면 능력치가 눈에 띄게 올라가" | Step 완료 후 "참고 가이드" 즉시 생성 |
| 카카오페이지 기다리면 무료 | ₩0 or ₩500~₩1,000 | "기다리기 싫으면 결제" | Step 1 무료, Step 2는 결제 |

**공통 심리 메커니즘:**
```
① 즉각적 만족 — 구매 후 바로 결과가 눈에 보인다
② 작은 단위 결제 — "전체 패키지"가 아니라 "이번 한 번만"
③ 반복 구매 유도 — 좋으면 자연스럽게 다음 스텝으로
④ 소셜 공유 — 이모티콘처럼 결과물을 카톡으로 보내고 싶어진다
⑤ 손실 없음 감각 — "잘못 사도 ₩990이니까 뭐"
```

**Harness UX 설계 원칙 (게임 아이템 심리 적용):**
```
Step 1 무료 → "AI 사용 자가점검 결과: 🔴 관심이 필요한 패턴입니다"
              → "다음 Step에서 당신에게 맞는 전략을 알려드립니다"
              → [Step 2 시작하기 — ₩990]  ← 게임의 "다음 레벨" 심리
```

결과물이 **즉각적이고 구체적**이어야 한다. "AI 교육 가이드"가 아니라 "지금 당장 이번 주에 할 것 3가지" 형태로.

```
심리적 저항선 아래 가격의 조건:
→ 커피 한 잔 ₩4,500과 비교해도 "훨씬 싸다"
→ 망설임 없이 결제 버튼을 누를 수 있는 금액
→ "이게 효과 없어도 손해 봤다는 생각이 안 드는" 금액
→ 카톡으로 "나 이거 해봤는데 신기해" 보내고 싶어지는 금액
```

#### 데이터 연동 동적 가격 조정 로직 (CEO 지시)

```python
# 개념 로직 (실제 구현: ar_tracker + analytics 연동)
def get_next_step_price(step_number, conversion_rate):
    """
    conversion_rate = 이전 Step 완료자 중 다음 Step 진행 비율
    """
    base_price = PRICE_TABLE[step_number]  # 기본 가격표
    
    if conversion_rate >= 0.60:  # 60% 이상 → 다음 단계 10~15% 인상 검토
        return base_price * 1.10  # A/B 테스트 후 결정
    elif conversion_rate >= 0.40:  # 40~60% → 현행 유지 (stay)
        return base_price
    else:  # 40% 미만 → 프로모션 또는 인하 검토
        return base_price * 0.80  # "이번 주 특가" 형식으로 제공
```

| 전환율 구간 | 가격 액션 | 실행 방식 |
|---|---|---|
| ≥ 60% | +10~15% 인상 검토 | A/B 테스트 2주 후 결정 |
| 40~60% | **Stay (현행 유지)** | 기본 원칙 |
| 20~40% | -20% 한시 프로모션 | "이번 달 특가" 뱃지 부착 |
| < 20% | 해당 Step 콘텐츠 개선 우선 | 가격 문제 아닌 가치 문제 |

**추적 지표 (매주 점검):**

| 지표 | 정의 | 임계값 |
|---|---|---|
| Step 전환율 | Step N 완료 → Step N+1 결제 비율 | 40% 이상 유지 목표 |
| lock-in 진입율 | Step 4 진입 비율 (Step 1 기준) | 15% 이상 목표 |
| 완주율 | Step 8 완료 비율 (Step 1 기준) | 5% 이상 목표 |
| 바이럴 계수 | 공유 클릭으로 유입된 신규 Step 1 비율 | K ≥ 0.3 목표 |

| ₩9,900 | ~$7.2 | **12명** | **14명** |

> **Step당 천원 모델의 장점:** 구독자 수 대신 "스텝 소비량"이 KPI가 되어 전환 장벽이 낮아진다. 첫 결제를 "월 구독"이 아닌 "₩990 테스트"로 만들면 CTR이 올라간다.

### Phase별 재무 경로

**Phase 0 — Pretotyping (0 유료)**
- 목표: WTP 클릭률 측정 (결제 버튼 클릭 ÷ 방문자)
- 비용: 광고비 $20~30 1회
- 수익: $0
- 기간: 2주 내 완료 목표

**Phase 1 — Pilot (5~10명 유료)**
- Step 모델: ₩990/스텝, 풀코스 ₩5,900
- 구독자 10명 × 5스텝 소비: $36 → LLM 비용 $60 미달 (광고비 없을 경우)
- 구독자 20명 × 5스텝: $72 → **BEP 달성 가능**
- KPI: 4주 내 유료 첫 결제 1건

**Phase 2 — Growth (30명 이상)**
- 월 스텝 소비 500스텝: 수익 $362, 비용 $100 → 순수익 $262
- 바이럴 입소문 시작 시점

## 5. 고객 획득 비용(CAC) 계획

### 채널 선택 원칙

- 부대표 또는 대표의 지인망에 직접 결제 요청 금지 (CLAUDE.md §Never)
- 수익은 마케팅을 통해 유입되는 **익명 고객**에서만 인정
- 외부 발행·광고 전 `legal_review_approve` + `qa_clear` 필수 (CLAUDE.md §Must)

### 채널별 CAC 시뮬레이션

**채널 A — 인스타그램 / 페이스북 유료 광고**

| 항목 | 값 |
|---|---|
| 타겟 | 30~45세 초·중등 자녀 학부모, 교육 관심 |
| 예산 | $20~30 (pretotyping 1회) |
| 예상 도달 | 1,000~3,000명 |
| 클릭률 목표 | 2~5% → 20~150 클릭 |
| 랜딩→결제 버튼 클릭률 | 10~20% → 2~30 WTP 측정 |
| 예상 CAC | $1~15/명 |
| 법률 제약 | 표시광고법 — "베타 참여" 문구 + QA `qa_clear` 필수 |

"Step 1 무료 체험" 광고 카피: **"AI 시대, 내 아이 어떻게 키울지 — 1분 자가점검 무료로 해보기"**

**채널 B — 맘카페·육아 커뮤니티 (Organic)**

| 항목 | 값 |
|---|---|
| 타겟 | 네이버 맘카페, 베이비트리, 맘스홀릭, 클리앙 육아 게시판 |
| 비용 | $0 (글 직접 게시) |
| 예상 도달 | 100~500명 (게시글 당) |
| 제약 | 광고성 글 금지 — "설문 참여 요청" 형식만 가능 |
| CAC | $0 (시간 비용만) |

**채널 C — 유튜브 쇼츠 / 릴스 (Organic)**
- 장기 브랜드 구축. 단기 전환엔 부적합. Phase 1 이후 병행.

**채널 D — VP 소셜 / 지인 관찰 (Product Insight만)**
- 부대표 생활권 Pain Point 관찰 → WTP 인터뷰 대상 모집에만 활용
- 결제 유도 금지 (CLAUDE.md §Never)

### 50명 달성 타임라인

| 단계 | 기간 | 목표 | 채널 | 비용 |
|---|---|---|---|---|
| Pretotyping | 2주 | WTP 클릭률 측정 | 인스타 광고 | $20~30 |
