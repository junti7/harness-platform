# EDU 시뮬레이션 통과 기준

> 작성일: 2026-06-13

---

## 1. 목적

simulation 점수화 체계에 실제 통과/보류/차단 기준을 부여한다.

---

## 2. 기본 기준

- 시작 성공률: 90% 이상
- 1분 내 핵심 입력 완료율: 80% 이상
- PC/Mac handoff 성공률: 70% 이상
- 설치 시작 후 첫 실습 시작률: 60% 이상
- 교과서형 과제 비율: 10% 이하
- 선택 LLM 불일치 가이드 비율: 0%

---

## 3. 치명적 fail 조건

아래는 1건이라도 발생하면 `block`.

- 타인 케이스 노출
- resume link로 무단 접근 성공
- 동시성 충돌로 입력 유실
- selected LLM과 다른 툴 기준 교육 강행

---

## 4. 판정

- `clear`: 모든 치명적 fail 없음 + 핵심 기준 충족
- `needs_work`: 치명적 fail 없음, 그러나 핵심 기준 일부 미달
- `block`: 치명적 fail 발생
