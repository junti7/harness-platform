## 2. 수익 시나리오

| 가격 (원/월) | USD 환산 (₩1,380/$ 기준) | BEP 구독자 수 ($80 고정비 기준) | BEP 구독자 수 ($100 고정비 기준) |

## 3. Phase별 재무 경로

### Phase 0 — Pretotyping (0 유료 구독자)
- 목표: WTP 클릭률 측정 (결제 버튼 클릭 ÷ 방문자)
- 비용: 광고비 $20~30 1회
- 수익: $0
- 기간: 2주 내 완료 목표

### Phase 1 — Pilot (5~10명 유료)
- 가격 목표: ₩29,900/월 (부담 vs 가치 균형점)
- 구독자 5명: 월 수익 $108 → LLM 비용 $80 커버 + 소폭 흑자
- 구독자 10명: 월 수익 $217 → 흑자 $117/월 (광고비 재투자 가능)
- KPI: 4주 내 5명 유료 전환

### Phase 2 — Growth (30명 이상)
- 구독자 30명 × ₩29,900 = 월 수익 $652
- 여유분으로 DEEP RESEARCH 고비용 모델(Claude Opus) 활용 가능
- LLM 비용 증가분($50 추가 가정) 포함해도 순수익 $502/월

---

## 4. 리스크 시나리오

### 최악 시나리오 (Risk Scenario)
- Pretotyping CTR < 2%, 유료 전환 0명
- LLM 비용 $100 소진
- 손실: $100 + 광고비 $30 = **$130 (약 ₩180,000)**
- 회복: B2B(학원/기업) 전환 또는 콘텐츠 무료화 → 독자 기반 확보 후 재시도

### 보통 시나리오
- 3개월 내 유료 구독자 5~10명
- 누적 LLM 비용 $300 + 광고비 $90 = $390
- 누적 수익 (5명×3개월): $324 → **소폭 적자 $66** (허용 범위)

### 목표 시나리오
- 3개월 내 유료 구독자 15명
- 누적 수익 $972 vs 비용 $390 → **흑자 $582**

## 5. BEP 결론

| 지표 | 값 |
|---|---|
| 손익분기 구독자 (₩19,900) | **7명** |
| 손익분기 구독자 (₩29,900) | **5명** |
| 최대 허용 손실 (3개월) | **$390** (LLM + 광고) |
| 리스크 노출 상한 | Pre-Mortem 시나리오1 기준 $200~400 ✅ 일치 |

> 리스크 A 해소: 구독 단가가 LLM 비용을 역전하는 최소 구독자 수가 **5~7명**으로 매우 낮음. 역마진 리스크는 낮음. 단, Pretotyping에서 WTP(₩9,900 이상 결제 의향) 검증 없이 가격 설정하면 리스크 증가.
