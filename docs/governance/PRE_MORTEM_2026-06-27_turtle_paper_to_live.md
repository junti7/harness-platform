## 의사결정 요약

현재 Alpaca 페이퍼 계좌는 -5.76% 상태이며, 진단 결과 **5개 구조적 결함(F1 롱전용 / F2 상관집중 /
F3 2×리스크 / F4 휩쏘 재진입 / F5 장부 불일치)** 이 확인됐다. 본 Pre-Mortem은 "이 결함들을 고치지 않고
실자본으로 넘어갈 경우" 최악 시나리오를 분석한다. **권고는 P0·P1 선행 수정 후 진입.**

## 시나리오 1 — 동시 손절 폭탄 (상관집중 + 2× 리스크 결합)

> "반도체 하루 급락에 전 종목이 같은 날 손절, 실손실이 계획의 4~8배"

| 항목 | 내용 |
|---|---|
| **시나리오** | F2(상관집중) + F3(2×리스크)가 결합. SMH·SOXX·MU·TSM처럼 동일 팩터 4종목 보유 중 반도체가 하루 -5% 급락 → 전부 2N 손절 동시 발동. 게이트는 종목당 "0.995%"로 통과시켰으나 실제 종목당 ~2%, 4종목 동시 = **하루 -8% 계좌 손실** |
| **발생 확률** | **35~45%** — 6/08에 이미 SMH·SOXX·MU·TSM 동시 손절이 실제로 발생. 반도체 단일 팩터 북에서는 흔한 사건 |
| **최대 손실** | 실자본 기준 단일 세션 -8%. 연속 발생 시 2주 -15~20% |
| **회복 가능성** | **중간** — 전략은 살아있으나, 외국환 한도($100k/년) 내 자본을 빠르게 소진. 재진입 여력 축소 |
| **Mitigation** | ① P0(2N 리스크 재측정)으로 종목당 ≤1% 강제 ② P1(상관 한도)로 동일팩터 동시보유 상한 ③ 실자본 진입 전 `turtle_gate_clear`에서 포트폴리오 합산 리스크 점검 |
| **Detection Trigger** | 단일 세션 계좌 손실 > 3% OR 동일 팩터 동시 손절 ≥ 2종목 |

## 시나리오 2 — 장부 불일치로 실손실 미인지 (F5)

> "봇은 SMH를 보유 중이라 믿는데 브로커는 18일 전 청산. 실자본에서 P&L을 모른 채 운용"

| 항목 | 내용 |
|---|---|
| **시나리오** | F5 그대로 실자본 이전. `trading_diary`·`positions.json`이 브로커 현실과 desync. 여러 스크립트가 한 계좌를 통합원장 없이 건드려, 손절 미기록·유령 포지션·중복 주문 발생. 리스크 매니지먼트가 잘못된 상태를 근거로 판단 |
| **발생 확률** | **50~60%** — 현재 페이퍼에서 **이미 발생 중인 확정 사실**(진입 2건/청산 0건 기록, SMH·SOXX 18일째 stale) |
| **최대 손실** | 실손실의 지연·은폐. 중복 주문으로 의도외 포지션. Kill Criteria·BRM 리스크 레지스터 오작동 → 손실 한도 돌파를 늦게 인지 |
| **회복 가능성** | **높음(수정 시)** — 통합 원장 + 일일 reconcile로 구조적 해소 가능. 단, 실자본에서 desync 상태로 굴리면 신뢰 회복까지 손실 누적 |
| **Mitigation** | ① P0(통합 원장 + `update_json_atomic` 락 + 브로커 일일 reconcile) **실자본 진입의 절대 선행조건** ② reconcile 불일치 시 자동 매매 정지(circuit breaker) |
| **Detection Trigger** | diary/state ↔ 브로커 포지션 불일치 1건 이상 OR 청산 이벤트 미기록 감지 |

## 시나리오 3 — 횡보장 휩쏘 누적 출혈 (F1 + F4)

> "추세 없는 반도체에서 롱 전용 봇이 매주 가짜 돌파를 사고 손절, 천천히 말라죽는다"

| 항목 | 내용 |
|---|---|
| **시나리오** | F1(롱 전용) + F4(추세필터 부재). 횡보 지속 시 매 상방 돌파를 사고 → 반락에 손절 → 다시 사기 반복. 숏은 못 잡아 하락 추세 수익도 0. 큰 사고 없이 수수료·슬리피지·휩쏘로 분기 -10~15% 잠식 |
| **발생 확률** | **40~50%** — 반도체가 현 횡보 국면을 1~2분기 지속할 경우. TSM 2회 연속 손절이 이미 전조 |
| **최대 손실** | 분기 -10~15%, 큰 추세 부재 시 BEP 회복 불가 |
| **회복 가능성** | **중간** — 추세 복귀 시 회복되나, 그때까지 자본·심리(CEO 신뢰) 소모 |
| **Mitigation** | ① P1(롱·숏 대칭 **또는** 200일선 추세필터로 횡보장 진입 억제) ② P2(직전 돌파 이익 시 다음 돌파 스킵) ③ 분산(무상관 자산)으로 단일팩터 횡보 의존 축소 |
| **Detection Trigger** | 4주 롤링 승률 < 30% OR 동일 종목 연속 손절 ≥ 2회 |
