### 핵심 전제
"부모의 심리적 의지가 자녀보다 100배 강하다"
→ 따라서 초·중등 자녀 학부모가 1차 타겟 고객
→ 부모를 설득하면 아이는 따라온다

### "Segment of One" 개인화 컨설팅
- 모든 고객이 "이건 나만을 위한 것"이라고 느껴야 함
- 인테이크 폼(15분) → LLM 분류 + 솔루션 생성 → 개인화 재작성 → 발송
- "000 부장님 케이스", "000 학년 아드님 상황" 등 이름·상황 명시

- 구독자 확장 후 ElevenLabs 음성, HeyGen 영상 단계적 추가

Track A: 직장인 업무 자동화 — 고민 청취 → 맞춤 솔루션 (15분 인테이크 폼)
Track B: 학부모 자녀 사례 — 구체적 상황 의뢰 → 연구 기반 맞춤 처방
Track C: Excel 신주단지 증후군 해방 — 3단계 심리적 전환 컨설팅
Track D: 대기업 바이브코딩 직장인 — "pip 차단 환경 생존법" + "AI 지시 설계법"

## 당신이 Red Team으로서 검토해야 할 5가지

1. **가장 강력한 비판점 3가지**: 이 사업 모델에서 가장 위험한 가정, 약점, 또는 잘못된 전제는 무엇인가? 실제로 실패할 가능성이 가장 높은 이유를 분석하세요.

2. **"Segment of One" + 스케일 모순**: "완전 개인화"와 "자동화 파이프라인으로 90% 처리"는 장기적으로 양립 가능한가? 이 모순을 성공적으로 해결한 실제 사례가 있는가?

3. **ChatGPT 직접 경쟁**: 학부모나 직장인이 ChatGPT에게 직접 물어보면 무료로 개인화된 답을 얻을 수 있다. 이 경쟁자 대비 Harness의 차별점이 지속 가능한가? 어떤 조건에서 지속 가능하고 어떤 조건에서 붕괴하는가?

4. **WTP 현실성**: 한국 학부모·직장인이 월 ₩29,900~₩59,000을 "AI 교육 컨설팅 구독"으로 지불할 의향이 있는가? 유사 서비스(직장인 자기계발, 교육 상담)의 실제 국내 WTP 데이터와 비교 분석해주세요.

5. **데이터 신뢰도 확인**: 아래 3개 데이터 소스의 신뢰도를 평가하고, 가능하면 원문을 직접 확인해주세요.
   - BFI WP 2025-144 (Becker Friedman Institute Working Paper) — 실제 존재하는 논문인가?
   - 나우앤서베이 2023 한국 직장인 AI 사용 73.9% — 현재 시점에서도 유효한 수치인가?
   - ElevenLabs API 세션당 비용 $1~3 — 실제 현재 요금과 맞는가?
