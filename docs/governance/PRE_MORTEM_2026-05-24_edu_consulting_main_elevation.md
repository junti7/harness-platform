## 의사결정 요약

Physical AI 뉴스레터와 B2I 투자 엔진을 부차적 역할로 전환하고, **"AI 의존을 막고 잠재력을 키우는 학부모 상시 인텔리전스 자문"**을 Harness의 메인 수익 모델로 격상한다. 1차 타겟: 초·중등 자녀를 둔 학부모. 수익 모델: 진단 + advisory retainer 구독. DEEP RESEARCH(데이터 수집 파이프라인) = 조사이자 제품 본체.

## 시나리오 1 — 학부모 WTP(지불의향) 불충분

> "FOMO는 있지만 실제로 돈은 내지 않는다"

| 항목 | 내용 |
|---|---|
| **시나리오** | 학부모의 AI 교육 불안은 실재하지만, 이에 대한 결제는 발생하지 않는다. 유튜브·맘카페·무료 toolkit이 이미 충분히 느껴지고, 추상적 "AI 활용 가이드"의 유료 가치를 체감하지 못한다. Pretotyping CTR < 2%, 인터뷰 응답의 60%+ "무료면 써볼게요" |
| **발생 확률** | **40~50%** — 스타트업 1차 실패의 가장 흔한 원인. 내러티브와 실제 결제 의향 간 간극은 항상 생각보다 크다 |
| **최대 손실** | 3~6개월 운영비($200~400) + 기회비용(뉴스레터·B2I 디코스트) + 방향 전환 비용 |
| **회복 가능성** | **중간** — B2B(학교·학원·기업) 전환 가능. 단, 영업 구조·타겟·커리큘럼 재설계 필요 (1~2개월) |
| **Mitigation** | ① Pretotyping을 Pilot보다 **먼저** 실행해 WTP를 실제 클릭·결제로 검증 ② 가격 실험: 무료 진단 → 유료 retainer 전환율 측정 ③ 인터뷰 5~10명 — "얼마면 낼 것인가" 구체적 수치 질문 |
| **Detection Trigger** | Pretotyping 결제 클릭률 < 2% OR 인터뷰 WTP 응답 평균 < 월 5,000원 OR Pilot 4주 후 유료 전환 의향 < 20% |

## 시나리오 2 — 경쟁사 선점 (대형 플레이어 한국어 진입)

> "우리가 만들기 전에 OpenAI 또는 국내 대형 EdTech가 같은 각도로 들어온다"

| 항목 | 내용 |
|---|---|
| **시나리오** | OpenAI의 Korean Khanmigo-equivalent, 엘리스·코드잇·클래스101 중 한 곳이 "부모를 위한 AI 교육 코치"를 대규모로 출시. 또는 "AI 의존 방지 학습법" 각도를 누군가가 먼저 점유. Harness의 3중 해자 중 핵심(반의존 포지셔닝)이 무력화 |
| **발생 확률** | **20~30% (12개월 내)** — AI 교육 시장은 투자 급증 중. 대형 플레이어의 피벗 속도는 예측 불가 |
| **최대 손실** | 시장 포지션 붕괴 + 차별화 없는 "me-too" 위치로 전락. 회복까지 새 각도 탐색 3~6개월 |
| **회복 가능성** | **낮음~중간** — OpenAI 규모와 경쟁은 불가능. 단, Harness는 한국 학부모 맥락 + VP 현장 감각 + 멀티LLM 커스텀 인텔리전스로 초미세 틈새 가능 |
| **Mitigation** | ① Class 5(경쟁 인텔리전스) 상시 레이더 — 채용공고·광고 라이브러리·투자 발표 모니터링 ② **빠른 선점** — pretotyping → pilot을 12개월이 아니라 6~8주 내 실행 ③ "한국 학부모 커뮤니티 밀착"이라는 대형 플레이어가 복제 못 하는 맥락 강화 |
| **Detection Trigger** | 경쟁사 채용공고에 "AI tutoring for Korean parents" 관련 직군 등장 OR 광고 라이브러리에 동일 키워드 캠페인 등장 → 즉시 CEO 보고 |

## 시나리오 3 — 학원법/교육서비스업 규제 적용

> "개인화 교육 자문이 법적 규제 대상이 된다"

| 항목 | 내용 |
|---|---|
| **시나리오** | 한국 학원법(학원의 설립·운영 및 과외교습에 관한 법률)이 Harness의 "우리 아이 학습 전략 진단·자문" 서비스를 **과외교습** 또는 교육서비스업으로 판단. 신고·시설·자격 요건 부과. 또는 교육 효과 미달 시 소비자보호법 문제 |
| **발생 확률** | **15~25%** — 콘텐츠 구독 형태이면 낮지만, "맞춤 진단"이 포함되면 회색지대 |
| **최대 손실** | 서비스 모델 재설계(맞춤 자문 → 순수 콘텐츠 구독), 사업자 신고 비용, 교육 효과 관련 disclaimer 의무화 |
| **회복 가능성** | **높음** — 서비스를 "정보 제공 구독"으로 포지셔닝 전환하면 학원법 적용 가능성 대폭 낮아짐 |
| **Mitigation** | ① **외부 변호사 자문** — 학원법 적용 여부 정식 검토 (외부발행 전 필수) ② 1:1 맞춤 자문보다 **"콘텐츠 구독 + 참고 가이드"**로 시작 ③ 효과 수치·진단 결과의 "보장" 표현 절대 금지 ④ 서비스 소개에 "정보 제공 목적, 전문 상담 대체 불가" 명시 |
| **Detection Trigger** | 유사 서비스 대상 학원법 적용 판례 등장 OR 교육부/지자체 에서 AI 교육 서비스 신고 의무화 행정지침 발표 |

## 시나리오 4 — "AI가 만든 AI 교육 콘텐츠" 신뢰 위기

> "LLM이 자녀 학습 조언을 준다는 아이러니가 역풍이 된다"

| 항목 | 내용 |
|---|---|
| **시나리오** | ① LLM 생성 교육 콘텐츠의 오류(잘못된 학습법 권고, 연령 부적합 조언)로 학부모 불신 확산 ② "AI 의존 방지를 AI한테 물어본다"는 아이러니가 SNS에서 바이럴화·조롱 대상 ③ 경쟁사나 언론이 "AI가 자녀 교육 개입" 리스크를 부각 |
| **발생 확률** | **25~35%** — 교육 분야는 신뢰가 핵심. 실수 한 번의 파급이 타 분야보다 크다 |
| **최대 손실** | 브랜드 평판 회복 불가 수준 손상, 구독 취소 급증, 언론 노출 시 최악. 서비스 종료 가능성 |
| **회복 가능성** | **낮음** — 특히 자녀 교육 관련 신뢰 붕괴는 회복이 극히 어려움 |
| **Mitigation** | ① **VP 검토 필수(모든 콘텐츠)** — "이 내용이 우리 아이에게 적용 가능한가" 학부모 관점 1차 심사 ② 학습과학 이론 인용 시 **출처 명시** (Class 1 논문 기반) ③ 개인화 진단에 "AI 생성 참고 자료, 전문가 상담 병행 권장" 항시 표기 ④ 아이러니 선제 해소: "저희는 AI를 활용하되, 최종 판단은 부모와 자녀가 합니다"를 브랜드 메시지로 |
| **Detection Trigger** | 후기/댓글에서 "AI가 틀렸다" 사례 3건 이상 누적 OR SNS 조롱성 바이럴 첫 등장 → 즉시 해당 콘텐츠 수정·공지 |

## 시나리오 5 — 집중력 분산 → 교육·뉴스레터·B2I 모두 미완

> "메인 격상을 선언했지만 실제 리소스 배분은 바뀌지 않는다"

| 항목 | 내용 |
|---|---|
| **시나리오** | 교육 메인 격상을 선언했지만: 뉴스레터 파이프라인은 계속 부분 가동, B2I IBKR 준비(AR-026 blocked)도 완전 중단 아님, DEEP RESEARCH 착수도 "설계"에 머물고 실제 수집은 시작 안 됨. 3개월 후 세 방향 모두 진척 없는 "좀비 사업" 상태 |
| **발생 확률** | **30~40%** — 자원 재배분 결정을 명시하지 않으면 관성이 이긴다 |
| **최대 손실** | 6~12개월 표류, CEO·부대표 소진, LLM 비용 지속 소모, 외부 관찰자(잠재 고객·투자자) 신뢰 손상 |
| **회복 가능성** | **높음** — 조기 발견 시 리소스 재배분 결정으로 즉시 교정 가능 |
| **Mitigation** | ① **명시적 deprioritization 결정**: 뉴스레터를 "최소 유지(주 1회 자동 발행)" 또는 "일시 중단"으로 CEO 공식 결정 ② B2I는 BOK 회신 대기이므로 실질 개입 없음 — "대기" 명시 ③ 교육 사업에 **주간 OKR** 설정: 4주 내 pretotyping 완료, 8주 내 pilot 완료 |
| **Detection Trigger** | 4주 경과 후 pretotyping 미완료 OR 8주 경과 후 유료 pilot 참가자 0명 → CEO 긴급 리뷰 |

**통과 조건 (이 Pre-Mortem이 말하는 "안전한 진입" 조건):**

1. ✅ G1 밑작업 전에 **Pretotyping(가짜문) 먼저** — WTP 시나리오1 조기 검증
2. ✅ 외부 발행 전 **학원법 전문가 의견** 확보 — 시나리오3
3. ✅ **VP 콘텐츠 검토 + 출처 명시** 시스템 구축 후 고객 노출 — 시나리오4
4. ✅ CEO가 **뉴스레터 최소화 결정** 명시 — 시나리오5
5. ✅ **경쟁 모니터링 레이더**(Class 5) 즉시 가동 — 시나리오2
